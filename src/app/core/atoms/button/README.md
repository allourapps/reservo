| Name | Default | Type | Description |
|------|---------|------|-------------|

| **label** | | string | |
| **type?** | 'raised' | 'raised' &#124; 'flat' | |
| **icon?** | | string | the name of the icon |
| **color?** | | string | |
| **disabled?** | false | boolean | |
| **children?** | | any | |
| **style?** | | Object | |
| **iconProps?** | | Object | |
| **className?** | | string | |
